Detailed Description

Calculates flux out of a storage as a linear function of its volume.

This connection serves the same purpose as the old kinematic_wave connection, but the parameters are easier to explain

\[ q = \frac{V - V_{residual}}{t_r} \]

where:

  • \(V\) The actual volume of water stored in source
  • \(V_{residual} [m^3]\) The volume of water not flowing out (default = 0) \(t_r\) is the residence time in the source.
